# Kilo2Pounds


## Basico convertidor de unidades entre kilogramos y libras

## Instalacion 

```
npm install kilo2pounds

```

## Uso

```
let converter = require('kilo2punds')

converter.toKilos(250) //113.40
converter.toPounds(100) //220.46

```
